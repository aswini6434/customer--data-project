# Customer Data Analysis

## Overview
This repository contains a **sample customer dataset** along with data cleaning, analysis, and visualization. It can be used to explore customer behavior, spending trends, and segmentation.

## Dataset Description

| Column Name       | Description                                                                 |
|------------------|-----------------------------------------------------------------------------|
| CustomerID        | Unique ID assigned to each customer                                         |
| FirstName         | Customer’s first name                                                       |
| LastName          | Customer’s last name                                                        |
| Email             | Customer’s email address                                                    |
| Gender            | Gender of the customer (Male/Female/Other)                                  |
| Age               | Age of the customer in years                                                |
| City              | City where the customer resides                                             |
| RegisteredDate    | Date when the customer registered                                           |
| NumberOfOrders    | Total number of orders made by the customer                                 |
| TotalSpend        | Total amount spent by the customer (in currency)                            |

## Project Objectives
- Perform "data cleaning" (handle missing values, duplicates, etc.)
- Conduct "Exploratory Data Analysis (EDA)"
- Create "visualizations" to show:
  - Age distribution  
  - Spending trends by city  
  - Relationship between orders and total spend
  -  customers registration
  -  gender distribution
  -  Top customers by spend
